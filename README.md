# 📖 Project Overview

The **AQI vs Health Impact Prediction** project uses Machine Learning to analyze how air quality affects human health. By leveraging environmental data and Air Quality Index (AQI) measurements, the model predicts the potential health impact associated with varying pollution levels.

The project involves comprehensive data preprocessing, exploratory data analysis, feature engineering, model training, hyperparameter tuning, and performance evaluation to develop an accurate predictive model.

---

# 🎯 Objectives

- Analyze the relationship between AQI and health impact.
- Clean and preprocess environmental datasets.
- Train a Machine Learning model for prediction.
- Improve prediction accuracy through hyperparameter tuning.
- Evaluate model performance using standard ML metrics.
